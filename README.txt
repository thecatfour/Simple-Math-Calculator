This program allows the user to input a math expression and get its result.

"ApplicationMain.java" is the version with a GUI.
"ApplicationFrame.java" sets up the GUI used by the main program.

"ConsoleMain.java" is the version that operates in a console window. It displays additional information in the event of an error.

"UnitTestClass.java" allows unit tests of the program. Read "UnitTestREADME.txt" for more information.

"SimpleMathParser/" contains the class files used by the program to parse the input into an expression and then find the answer.
